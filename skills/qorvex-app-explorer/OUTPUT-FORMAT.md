# Output Format Templates

Templates for each output artifact produced by the app explorer.

## screen-hierarchy.md

```markdown
# Screen Hierarchy: [App Name]

Bundle ID: [com.example.app]
Explored: [YYYY-MM-DD]
Device: [Simulator/Physical] — [Device Name]

## Screen Tree

- **Home** (Tab: Home)
  - **Item Detail** (tap list cell)
    - **Edit Item** (tap "Edit" nav button)
  - **Search Results** (tap search bar, enter query)
- **Profile** (Tab: Profile)
  - **Edit Profile** (tap "Edit" button)
  - **Settings** (tap gear icon)
    - **Notifications** (tap "Notifications" cell)
    - **Privacy** (tap "Privacy" cell)
    - **About** (tap "About" cell)
- **Create** (Tab: + button / modal)

## Screen Details

### Home
- **Navigation:** Tab bar item 1
- **Elements:** [count] interactive, [count] static
- **Key elements:** Search bar, item list (scrollable), filter button
- **Screenshot:** screenshots/home.png

### Item Detail
- **Navigation:** Tap any list cell on Home
- **Elements:** [count] interactive, [count] static
- **Key elements:** Title, description, action button, share button
- **Back:** Nav bar back button → Home
- **Screenshot:** screenshots/item-detail.png
```

## user-flows.md

```markdown
**Steps:**

1. [Screen] — [Action description]
2. [Screen] — [Action description]
3. ...

**End state:** [What the user sees when done]
**Script:** scripts/flow-[name].sh

---

## Flow: Create New Item

**Purpose:** User creates a new item and saves it
**Entry point:** Home screen
**Steps:**

1. **Home** — Tap the "+" button in the tab bar
2. **Create Item** — Enter title in "Title" field
3. **Create Item** — Enter description in "Description" field
4. **Create Item** — Tap "Category" picker, select a category
5. **Create Item** — Tap "Save" button
6. **Home** — Verify new item appears in the list

**End state:** Home screen with new item visible at top of list
**Script:** scripts/flow-create-item.sh
```

## scripts/ (automation scripts)

Each script is a standalone bash file that replays a user flow via qorvex.

```bash
#!/bin/bash
# Flow: [Flow Name]
# Generated by qorvex-app-explorer
# Prerequisites: qorvex session active, target set to [BUNDLE_ID]

set -euo pipefail

SCREENSHOT_DIR="/tmp/app-explorer/[bundle-id]/screenshots"
mkdir -p "$SCREENSHOT_DIR"

# Step 1: Verify starting screen
qorvex wait-for -l "Home" -o 5000
qorvex screenshot 2>/dev/null | base64 -d > "$SCREENSHOT_DIR/flow-start.png"

# Step 2: Tap create button
qorvex tap -l "Create"
qorvex wait-for -l "Title" -o 5000

# Step 3: Fill in title
qorvex tap -l "Title"
qorvex send-keys "My New Item"
qorvex swipe down  # dismiss keyboard

# Step 4: Save
qorvex tap -l "Save" -T Button
qorvex wait-for -l "Home" -o 5000

# Step 5: Verify
qorvex screenshot 2>/dev/null | base64 -d > "$SCREENSHOT_DIR/flow-end.png"
echo "Flow complete."
```
