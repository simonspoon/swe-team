Workflow 1 — Task Lifecycle

A limbo task moves through 7 stages from captured to done. The project-manager dispatches a specialist subagent at each forward transition. Five rollback paths return a failing task to an earlier stage. A trivial task can take the fast-path bypass, skipping researcher and red-team.
