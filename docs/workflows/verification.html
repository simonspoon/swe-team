Workflow 3 — verification-orchestrator skill workflow

The verification-orchestrator skill runs a 4-step pipeline: detect the platform, check QA tool availability, route to the matching QA skill, and produce a report. A multi-platform project (Tauri, Electron, Capacitor) fans out so every platform is verified. A missing tool takes the SKIPPED path and the gap is noted.
